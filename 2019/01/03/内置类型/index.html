内置类型

基本类型

  • null
  • undefined
  • number
  • string
  • boolean
  • Symbol
    • es6新增类型
    • var s = Symbol()生成值，可接受任意值作为参数。如果用这种方式生成，那么Symbol(1) === Symbol(1)为false。其产生的值每个都是独一无二的。但是Symbol.for(1) === Symbol(1)为true。可类比为Java的intern()方法，如果全局中，已存在该变量那么直接返回该变量。
    • Object.keys()和Object.getOwnPropertyNames()不会返回包含Symbol的属性名，但有Object.getOwnpropertySymbols()方法可以返回。

对象

属于引用类型，使用过程中会遇到深拷贝和浅拷贝的问题。

判断类型

  • typeof
    • 基本类型，除了null会显示Object，其他都可以显示正确的类型。
    • 引用类型，只能显示Object
  • Object.prototype.toString.call(xx)可获得类似[object Type(如Array)]返回值
  • instanceof可返回Type。

相互转换

  • +运算符：如果都是数字，那么直接相加。否则转换成字符串类型再相加。

  • 其他运算符：转换成数字再运算，无法转换则返回NaN

  • 转boolean类型

    • 以下类型都会转成false：undefined、null、false、NaN、''、0。

  • 基本类型转换

    • toString()方法，都会转成字符串类型。
    • valueOf()方法，会返回原值，类型不变。

  • 对象转基本类型：

    • 转成字符串时候toString()方法优先调用，转成数字则valueOf()方法优先。

    • 有操作符，比如==等，优先调用valueOf()方法

    • toString()方法可传入参数转进制，null和undefined都没有该方法，但new String()可以初始化null和undefined。

    • Symbol.toPrimitive优先级最高，可接受参数，表示需要转换类型：

      • Number：乘法等算数运算
      • String：String(str)，${str}等情况
      • Default：+连接运算符
