python使用总结

发表于 2018-09-15 | 分类于 python
  1. 判断类型：type(obj).__name__ == 'dict' # list、dict、set
  1. django对查询出来的querySet对象转换成json：
  • 对象.字段 = json对象[“字段”]

  • 上述两种方法转成json对象后，存在的dict也会被序列化变成json字符串，所以需要调用json.loads方法。

3.时间操作：
时间格式一般有datetime、unix、OSI（yyyy-mm-ddTHH:mm:ss:fff+时区Z），主要是前两者的转化为多。

  • datetime转换成unix：

  • unix转换成datetime

  • datetime转换成OSI：datetime.isoformat()

  • datetime转换成str: datetime.strftime('%Y-%m-%d %H:%M:%S')
  • str转换成datetime: datetime.strptime(str, '%Y-%m-%d %H:%M:%S')
    4.类型转换
  • int()、float()、str()可以将数据转换对应类型
  • list()、set()也是如此

5.字典操作：

  • 判断长度为0：len(dict)==0
  • 判断字典是不是包含key：if d in ret.keys():
  • 合并字典：d1.update(d2)
  • 字典键：dict.keys()，字典值：dict.values(),遍历：dict.items()
  • get(key, default)方法不抛出异常
  • 删除元素：pop(key)

6.list操作：

  • 在末尾追加：list.append
  • 在任意位置插入：list.insert(index,value)
  • 统计次数：list.count(value)
  • append：添加元素；insert(index, element)：指定位置插入元素
  • index：根据值查找索引
  • remove：根据值删除；del list[index]：根据索引删除
  • extend：合并两个数组
  • sort()排序， reverse()倒序
    7.字符串操作：
  • 截断:s[first:last:step
  • 统计出现个数：s.count(ch)
  • 查找：find()和index()，找不到时候，前者返回-1，后者抛异常
  • list转字符串：'.'.join(['1','2','3'])，保证list是字符串数组
  • 字节数组转字符串：str(bytes, encoding="utf-8")，字符串转字节数组：str.encode(s)
  1. enumerate函数表示将元祖等转化成具有下标和值得可迭代元素，类似go的range。

9.函数式编程：
多用reduce：去两个数操作,map：对每个数操作，不用循环。
